# Nothing landing page
This project represents a landing page for *Nothing*, a british consumer electronics manufacturer. It showcases responsive design principles, intuitive UI, and dynamic components built with clean and reusable SCSS styles.

## Features
- **Responsive Design**: Optimized for various devices and screen sizes.
- **Intuitive UI**: Designed for a seamless user experience.
- **Reusable SCSS Styles**: Clean and modular SCSS ensures easy maintenance and scalability.

## Technologies Used
- **HTML**: For structuring the content of the page.
- **SCSS**: For styling with enhanced flexibility and reusability.

## Running the project locally
1. Clone
```bash
git clone https://github.com/yuriipodria/nothing_landing.git
cd nothing_landing
```

2. Switch to Node.js v14
```bash
nvm install 14
nvm use 14
```

3. Install dependencies
```bash
npm i
```

4. Start the development server
```bash
npm start
```
